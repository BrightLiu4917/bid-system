# 环境依赖

## 必需依赖

| 依赖 | 用途 | 说明 |
|------|------|------|
| Bash | 执行安装、检查、测试脚本 | macOS / Linux / WSL 均可 |
| Git | 版本管理、diff、审查 | 强烈建议目标项目先 `git init` |
| Codex App 或 Codex CLI | 主控 Agent | 负责架构、拆分、审查 |

## 推荐依赖

| 依赖 | 用途 |
|------|------|
| ripgrep (`rg`) | 快速搜索代码和文档 |
| Docker | 启动 MySQL、Redis、依赖服务 |
| curl | deepv4 OpenAI-compatible API 调用 |

## 后端常见依赖

| 技术 | 用途 |
|------|------|
| JDK 17+ | Java / Spring Boot 项目 |
| Maven 或 Maven Wrapper | 后端构建和测试 |
| MySQL 8 | 业务数据库 |
| Redis | 缓存、会话、分布式锁等 |

## 前端常见依赖

| 技术 | 用途 |
|------|------|
| Node.js | 前端构建 |
| pnpm / npm | 包管理 |
| Vue 3 / React | 前端框架 |

## 本地 Agent 可选依赖

| 依赖 | 用途 |
|------|------|
| OpenCode | 本地代码执行 Agent |
| Ollama | 本地模型运行 |
| qwen3-coder | 本地代码模型 |

示例配置：

```text
.agent/local-agent.env
LOCAL_AGENT_COMMAND='opencode run'
PROJECT_TEST_COMMAND='cd api && ./mvnw test && cd ../vue && pnpm build'
```

## deepv4 可选依赖

deepv4 通过 OpenAI-compatible API 接入：

```text
DEEPV4_BASE_URL='https://api.example.com/v1'
DEEPV4_API_KEY='replace-with-your-key'
DEEPV4_MODEL='deepv4'
DEEPV4_REVIEW_COMMAND='bash scripts/providers/deepv4-openai-compatible.sh'
```

真实 key 只能写在：

```text
.agent/local-agent.env
```

不要提交到 git。

## 支持的项目形态

| 项目形态 | 支持情况 |
|----------|----------|
| Java / Spring Boot / MySQL | 优先支持 |
| Vue / React 管理端 | 支持通用前端流程 |
| api + vue monorepo | 支持 |
| Go / PHP / Node | 可安装控制系统，但需要补充对应后端 skill |
| gupo 项目 | 通过 gupo profile 支持专用 adapter |

## 不会自动做的事

- 不自动安装 JDK、Node、Maven、Docker。
- 不自动拉取 Ollama 模型。
- 不自动创建数据库。
- 不自动执行迁移 SQL。
- 不自动调用 deepv4，除非你配置并运行脚本。
