# UX 审查

## 必须读取
- `AGENTS.md`
- `docs/UX_RULES.md`
- `docs/PRODUCT_RULES.md`

## 流程
1. 明确用户入口和任务目标。
2. 梳理主流程和异常流程。
3. 检查加载态、空态、错误态、成功态、权限态。
4. 检查表单校验、重复提交和高风险确认。
5. 输出交互缺口和待确认问题。

## 输出
- 用户流程
- 状态设计
- 表单和校验
- 权限态
- 风险操作确认
- 待确认交互
