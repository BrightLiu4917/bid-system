# 前端测试

## 必须读取
- `AGENTS.md`
- `docs/FRONTEND_RULES.md`
- `docs/TESTING_RULES.md`

## 流程
1. 明确需要验证的用户行为。
2. 优先通过公开页面、组件接口或用户操作验证行为。
3. 覆盖加载态、空态、错误态、权限态和成功反馈。
4. 如项目已有测试框架，遵循既有命令和风格。
5. 无法自动化时，输出最小手动验证步骤。

## 输出
- 测试范围
- 测试命令
- 验证结果
- 未覆盖风险
