# 性能审查

## 必须读取
- `AGENTS.md`
- `docs/PERFORMANCE_RULES.md`
- `docs/DB_SCHEMA_RULES.md`

## 流程
1. 明确数据规模和访问频率。
2. 检查 SQL、索引、分页和 N+1。
3. 检查缓存、批量操作和事务范围。
4. 检查前端重复请求、大列表和渲染成本。
5. 输出优化建议和验证方式。

## 输出
- 性能风险
- 查询路径
- 前端渲染风险
- 优化建议
- 验证步骤
