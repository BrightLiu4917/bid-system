# 视觉质量检查

## 必须读取
- `AGENTS.md`
- `docs/UI_DESIGN_RULES.md`
- `docs/UX_RULES.md`

## 流程
1. 启动或定位前端页面。
2. 检查桌面和移动视口。
3. 检查加载态、空态、错误态、权限态和长文本。
4. 检查布局重叠、溢出、跳动和按钮可点击区域。
5. 输出截图或明确手动验证结果。

## 输出
- 检查页面
- 视口
- 发现问题
- 修复建议
- 剩余风险
