# UI 设计系统

## 必须读取
- `AGENTS.md`
- `docs/UI_DESIGN_RULES.md`
- `docs/FRONTEND_RULES.md`

## 流程
1. 识别既有组件、颜色、图标、间距和布局约定。
2. 优先复用既有设计系统。
3. 检查层级、密度、表格、表单、弹窗、抽屉和按钮主次。
4. 检查文本溢出、重叠、多视口和状态展示。
5. 输出设计风险和实现建议。

## 禁止
- 未确认引入新设计系统。
- 无意义装饰。
- 卡片套卡片。
- 文本溢出或遮挡。
